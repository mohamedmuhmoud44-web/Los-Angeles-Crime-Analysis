# Los Angeles Crime Analysis & Insights

## 📌 Description

This project presents an end-to-end data analysis of crime data in Los Angeles, aiming to uncover meaningful patterns, trends, and insights.

The workflow begins with data cleaning, where missing values are handled, duplicates are removed, and data types such as dates and time are properly formatted. Additional features are engineered from the dataset, including extracting hour, day, month, and time periods to enable deeper analysis.

Next, an exploratory data analysis (EDA) phase is conducted to understand the structure of the data, examine distributions, and identify key characteristics such as the most frequent crime types, the most affected areas, and general victim demographics.

The project then applies data visualization techniques using Matplotlib and Seaborn to clearly present findings. Visualizations include crime distribution by type, area, time (hour, day, month), as well as demographic patterns and heatmaps that reveal peak crime periods.

Finally, the project extracts key insights from the analysis, such as identifying high-crime areas, peak hours of criminal activity, the most common types of crimes, and temporal patterns in crime occurrence.
